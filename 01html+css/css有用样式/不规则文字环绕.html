近日，多名TikTok用户纷纷涌入小红书平台，这一现象迅速在网络上引发热议。令人意想不到的是，一些细心的美国网友发现，特斯拉创始人马斯克的母亲——梅耶·马斯克也悄然在小红书上开设了个人账号，这一消息瞬间在美国网友中炸开了锅，引发了广泛的热评和关注。为了更好地保护个人隐私和避免不必要的纷扰，梅耶·马斯克目前已将其小红书账号的评论功能设置为仅好友可见，关闭了公开评论功能。据搜索该账号的相关信息显示，梅耶·马斯克的第一条视频发布于2021年4月20日，那条视频也是她首次官宣入驻小红书的标志性内容。自入驻以来，她的账号就备受瞩目。据第三方数据监测平台显示，该账号的粉丝数已经突破了67万大关，尤其是在14日当天，涨粉数更是惊人地超过了4.4万，足见其影响力和受欢迎程度。 此外，梅耶·马斯克的账号主页还显示，她已经签约了知名的MCN机构——CAA创新精英文化经纪。通过查询企查查APP，我们可以了解到，这家MCN机构关联的公司为创新精英文化经纪（北京）有限公司，其注册资本高达1697.8万美元，实力雄厚。该公司成立于2005年9月，法定代表人及董事长为顾抒航。在对外投资方面，该公司持有上海一物影视文化合伙企业（有限合伙）50%的股份。
